# Cursor Landing Page

A modern, responsive landing page for Cursor—the AI-powered code editor that's transforming how developers work. This project showcases the features, testimonials, and value proposition of Cursor in a sleek, dark-themed design.

## Overview

This landing page is built to highlight Cursor's key features and capabilities. It includes sections dedicated to the AI Agent, autocomplete functionality, ecosystem integration, and real-world testimonials from industry leaders. The design emphasizes innovation and productivity with a premium dark aesthetic.

## Project Structure

```
Cursor/
├── index.html       # Main HTML markup and page structure
├── style.css        # Styling and responsive design
└── README.md        # This file
```

## Features

The landing page showcases several key sections:

- **Header & Navigation** - Sticky header with navigation menu, sign-in, and download buttons
- **Hero Section** - Compelling headline introducing Cursor with a call-to-action
- **Agent Features** - Highlights Cursor's AI Agent capability for turning ideas into code
- **Autocomplete** - Showcases the custom Tab model with striking accuracy
- **Ecosystem Integration** - Details on how Cursor integrates with GitHub, Slack, and other tools
- **Partner Logos** - Displays companies trusting Cursor daily
- **Testimonials** - Real quotes from industry leaders including Y Combinator, OpenAI executives, and top developers
- **Frontier Features** - Information about cutting-edge models, semantic search, and enterprise options
- **Changelog** - Recent updates and improvements to Cursor
- **Careers Section** - Invitation to join the Cursor team
- **Blog Highlights** - Recent articles and research from the team
- **Footer** - Comprehensive navigation links and company information

## Technologies Used

- **HTML5** - Semantic markup for structure
- **CSS3** - Modern styling with Flexbox and CSS Grid layouts
- **Responsive Design** - Mobile-friendly with adaptive layouts

## Styling Highlights

The CSS implements:

- **Dark Theme** - Premium dark background (#14120b) with light text
- **Grid Layouts** - Multi-column designs for testimonials, partners, and cards
- **Flexbox** - Flexible component arrangement throughout
- **Sticky Header** - Navigation that stays visible while scrolling
- **Rounded Elements** - Soft borders on images and buttons for modern look
- **Card-based Design** - Organized sections with consistent styling

## Customization

To customize this landing page:

1. **Update Content** - Edit the text, headings, and descriptions in `index.html`
2. **Replace Images** - Update image paths and add your own assets
3. **Modify Colors** - Change the color values in `style.css` to match your brand
4. **Add Links** - Update `href` attributes in navigation and CTA buttons
5. **Adjust Spacing** - Modify `padding` and `margin` values for different layouts

## Usage

Simply open `index.html` in your web browser to view the landing page. No build process or dependencies are required—it's pure HTML and CSS.

```bash
# Option 1: Open directly in browser
open index.html

# Option 2: Use a local server (Python)
python -m http.server 8000

# Option 3: Use with VS Code Live Server
# Install the Live Server extension and right-click "Open with Live Server"
```
